Torified hosting

Design documentation and code will come later, but the internetion is that every connection can only be made with Tor.
For now, you can see some scripts and configuration on Free Dumb Host's Github

The service is configured to only be accessible through tor, and servers are only able to make outgoing connections through Tor.
This means even if you hosted an insecure application, you simply cannot leak a real world IP address.

It's based off something like the grugqs Portal of Pi, with a lot of qemu and vlanning spiced through it.

Speaking of, if you do manage to break it, let me know I guess so I can patch it.
